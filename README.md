## General info
* Modern and simple drawing application
* Bug free [hopefully]
* Simple shapes
* Regular shapes (holding shift + simple shape)
* Freehand drawing
* Move objects after drawing
* Color and transparency settings
* Layers
* Canvas export as PNG
